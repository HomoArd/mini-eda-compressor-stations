# Analysis of Compressor Station Performance

## Project Overview
This project is a mini exploratory data analysis (EDA) of compressor station operations.
The goal is to analyze pressure, flow, and temperature indicators, identify unusual observations, and compare stations, regions, and operating modes.

## Objectives
- inspect the structure and quality of the data
- merge measurement data with station information
- create additional analytical features
- compare stations by average pressure and flow
- detect potential anomalies
- study the relationship between flow and pressure

## Tools
- Python
- pandas
- matplotlib

## Data
The project uses a small educational dataset with:
- station measurements: pressure, flow, temperature, date
- station information: mode, region, station type

## Project Structure
- data inspection
- merge and preparation
- feature engineering
- aggregations
- filtering
- visualization
- final conclusions
